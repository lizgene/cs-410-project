Exploring the Typekit Font Library

There are 5,000 fonts in the Typekit font library, but what do we know about them?

In particular, I was curious about the relationship between open source font availability and font usage.

  • How many open source fonts are currently available in the Typekit library?
  • How many foundries offer open source font options?
  • Are open source fonts more likely to be used than paid fonts?
  • What are the most popular fonts and foundries?

Climb aboard, font explorer! Because we are about to find out.
